void start (String songName, String songPath, num resId, Duration length, String cachedPath)

Starts this.

Performs an asynchronous call to music fetcher to start the Kuramoto loop.

Implementation

void start(String songName, String songPath, num resId, Duration length,
    String cachedPath) {
  fetcher.fetchLocalMusic(
      songName, songPath, resId, length, cachedPath);
  if (Platform.isAndroid || Platform.isIOS) {
    if(isPlaying) print("ERROR: THIS FLOW SHOULDN'T HAPPEN. TRIED TO PLAY WHEN ALREADY PLAYING. THIS CASE SHOULD HAVE CALLED NEXT INSTEAD OF START");
    else play(cachedPath);
  }
}
