processBeats method

void processBeats (List<Duration> newBeats)

Will take a list of durations between each beat and will process them according to when the last beat occurred or the last tempo change

Implementation

void processBeats(List<Duration> newBeats){ //note: cannot use now, because the beats could be in the future... and modifying duration respectively to now will be incorrect (not symmetrical)
  newBeats.forEach((beatTime) {
    if(lastTempoChange.isAfter(lastBeat)){
      beatTime = lastBeat.add(beatTime).difference(lastTempoChange);
      lastBeat = lastTempoChange;
    }
    Duration accelerated = new Duration(milliseconds: ((beatTime).inMilliseconds/totalMusicFactorChange).floor());
    lastBeat = lastBeat.add(accelerated);
    beats.add(lastBeat);
  });
  beats.removeWhere((beat) => beat.isBefore(DateTime.now()));  //remove outdated beats
}
