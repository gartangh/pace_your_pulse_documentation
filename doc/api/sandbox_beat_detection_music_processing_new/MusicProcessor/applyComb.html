applyComb method

int applyComb (List<num> x, num bpm, int frequency)

Apply a comb filter on the List x. With the given bpm; Using the given sampleFrequency and the sampleSampleFreq in the function itself. As this will influence on the distance between each sample in time. it returns an int, as best offset of sample that matches the given bpm.

Implementation

int applyComb(List<num> x, num bpm, int frequency){
  int sampleDistance = (frequency.toDouble()/window * 60.0/bpm).ceil();

  num bestSum = 0;
  int bestOffset = 0;
  for(int i = 0; i < sampleDistance; i++){
    num sum = 0;
    num samples = 0;
    for(int j = i; j < x.length; j += sampleDistance){
      sum += x[j];
      samples++;
    }
    sum /= samples; //to avoid that some sums are greater because they have 1 value more.
    if (sum > bestSum){
      bestSum = sum;
      bestOffset = i;
    }
  }

  return bestOffset;


}
