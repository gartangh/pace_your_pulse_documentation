convert(String inputFile, double startTime, double duration, double speedFactor, String outputFile) → Future
Convert the inputFile to an outputFile inputFile: the path where the input is saved startTime: the time where the output fragment should start duration: the duration of the input file that has to be converted, when -1 the whole audio file is converted speedFactor: the acceleration factor of the song, when 1.0 there is no change in speed outputFile: the path where the output is savsed
cutAndChangeSpeed(String inputFile, Duration startTime, double speedFactor, String outputFile) → void
This function converts an input file by cutting it into 2 halves around startTime. The first part remains the same, but the speed of the second part is changes according to speedFactor. The files are concatenated and written to outputFile.
