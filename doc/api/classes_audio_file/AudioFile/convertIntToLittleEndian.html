List<int> convertIntToLittleEndian (int input, int bytes)

input: a single integer value output: List of integers in that are stored Little Endian

Implementation

List<int> convertIntToLittleEndian(int input, int bytes) {
  List<int> output = List<int>(bytes);

  for (int i = 0; i < bytes; i++) {
    output[i] = ((input >> 8 * i) & 0xFF).toInt();
  }

  return output;
}
