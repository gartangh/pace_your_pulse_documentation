distanceString property

String distanceString

The distance as a formatted string.

Implementation

String get distanceString => _distance == null
    ? 'Unknown'
    : _distance <= 1000
        ? '${(_distance).toStringAsFixed(1)} m'
        : '${(_distance / 1000).toStringAsFixed(1)} km';
